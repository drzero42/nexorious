== External Database ==

To use an external database instead of in-cluster PostgreSQL, disable ALL three:
  controllers.postgresql.enabled: false
  service.postgresql.enabled: false
  persistence.postgresql-data.enabled: false
  nexorious.databaseUrl: "postgresql://user:pass@host:5432/dbname"

== Naming Note ==

This chart was designed to be installed as:
  helm install nexorious ./charts/nexorious

If you used a different release name, update nexorious.databaseUrl OR set:
  global.fullnameOverride: nexorious
