# Known issues

## IGDB ratings are shown incorrect
They should be shown with only a single digit following the dot.

## Steam Games ignore button breaks filter
1. Go to Steam Games
2. Filter the unmatched games by typing something in the search field
3. Click ignore next to a game
The list now goes from filtered to showing all games, even though the search term is still in the search field

## Search field on My Games has looking glass on top of text
In the search field on My Games there is a looking glass icon and the text "Search by title, genre, developer...". The looking glass is placed on top of the word "Search".
