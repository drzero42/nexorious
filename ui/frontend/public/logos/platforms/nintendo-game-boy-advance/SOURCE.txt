Game Boy Advance Logo
=====================

Source: Original geometric recreation (hand-created for this project).
Downloaded: 2026-06-24

Description: Game Boy Advance handheld silhouette: the landscape ('hamburger') body with a central screen, d-pad on the left and two buttons on the right.

This is an original, clean geometric drawing made for this project — not a
trace or copy of any brand asset file (the same approach as the Telltale mark
in ui/frontend/public/logos/storefronts/telltale/). It is free to use here.

The mark is a single fill-rule=evenodd path so inner details (screen, buttons,
counters) are theme-safe knock-outs:
- nintendo-game-boy-advance-icon-light.svg: black mark (#000000) for light backgrounds
- nintendo-game-boy-advance-icon-dark.svg:  white mark (#ffffff) for dark backgrounds
